# Antigravity Client (v1.0.20)

GUI-клиент и локальный прокси-сервер для интеграции и перенаправления запросов авторизации/API из **Antigravity 2.0** и **Antigravity IDE** на менеджер пула аккаунтов.

## 🏗 Архитектура запросов
```
IDE (VS Code / JetBrains) 
   --> Локальный прокси CLIента (порт 8047) 
   --> Antigravity-Manager (порт 8045 или 8055) 
   --> Google Backend
```

---

## 🚀 Что сделано в версии 1.0.18

1. **Новый дизайн блока подключения (Premium Banner)**:
   * Полностью переработан блок "Connect to IDE". Теперь он выглядит как премиальный плавающий баннер с эффектом glassmorphism и свечением. Кнопки выбора типа IDE сделаны шире и заметнее.

2. **Оптимизация отображения доступных моделей**:
   * Удалены громоздкие заголовки категорий. Теперь все ИИ-модели выводятся единым ровным и компактным списком (Grid). Категория модели теперь элегантно обозначается цветной точкой рядом с названием.

3. **Компактность статус-карточек и размеров окна**:
   * Карточки статуса, моделей и сервера вверху стали компактнее: убавлены отступы, а информация о тарифе и днях скомпонована в одну удобную строку.
   * Отключена авто-максимизация окна. Теперь приложение всегда запускается в компактном, идеально выверенном разрешении (840x840), где все блоки встают ровно и без переносов.

4. **Локализация**:
   * Добавлены отсутствовавшие переводы для статусов работы прокси ("Active" -> "Активен" и др.) в русском интерфейсе.

## 🛠 Запуск и разработка

Для запуска клиента в режиме разработки:
```bash
npm run tauri dev
```

Для сборки продакшн-версии:
```bash
npm run tauri build
```
